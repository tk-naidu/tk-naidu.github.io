User portal Welcome to the Insurance Portal Application guest user portal home! The purpose of this portal is to allow users to post feedback reviews of jobs done.

Server Admin Team

The server/portal side provides the following functionality.

  • Allows the addition or removal of a plumber, as well as signing up new call centre agents.
  • Allows for the creation of a new case as well as adding a new client if need be. As well as allocation of an available plumber to the case.
  • Generation of reports based on certain criteria as well as graphs for visual interpretations.



    • App team

    On the Mobile application side the following features were made available

  • A personalised "to-do" list is now sent to each plumber specifically, allowing no mix ups to take place.
  • when performing any job the work flow process has been made easier by utilizing smart phone capabilities such as allowing the scanning of barcode details and uploading pictures directly to the database for storage.
  • once a job is complete it gets removed from the list and is replaced by a new one if there is one available
